# Food-Group

### User Story
---
**AS** someone who wants to make a homemade meal
**I WANT** to see a recipe and where to buy the ingredients
**SO THAT** I can get everything I need with little stress

### Acceptance Criteria
---
**WHEN** the user searches for a Recipe **THEN** a list of recipes is shown.
**WHEN** the user selects a recipe **THEN** on another page the full recipe is displayed
Features on page: Recipe image, recipe card, list of ingredients, cooking instructions.
**WHEN** the user clicks on the link to the recipe **THEN** they are directed to an external recipe page.
**WHEN** the user clicks on the Google Maps link **THEN** they are directed to an external Google maps page.
